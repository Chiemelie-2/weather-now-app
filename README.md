
# **Weather Now APP** üå§Ô∏è

A simple and interactive **weather app** that shows the current weather and a 7-day forecast for any city worldwide. You can check the weather for any city of your choice, see hourly updates for up to 8 hours per day, and get weather information instantly using your browser.

---

## **Features** ‚úÖ

* Search weather for any city worldwide.
* Show 7-day weather forecast.
* Show temperature in Celsius and Fahrenheit.
* Detect your location using GPS.
* Display weather icons for clear visualization.
* Hourly updates (up to 8 hours per selected day).

---

## **Tech Stack / Tools** üõ†Ô∏è

* **Frontend:** HTML, CSS, JavaScript
* **APIs:** OpenWeatherMap API, [Nominatim OpenStreetMap](https://nominatim.org/) for geocoding
* **Deployment:** Vercel, GitHub Pages

1. Clone or download the repository:
